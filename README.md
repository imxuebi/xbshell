# xbtools使用说明

##xbtools使用方法 官方版

目前支持Debian，Ubuntu，termux的环境

使用指令：
1.
```Bash
apt install curl #Ubuntu/Debian
pacman -Sy curl #Arch
bash -c "$(curl https://shell.xb6868.com/xbtools.sh)"
```

2.
```Bash
apt install wget #Ubuntu/Debian
pacman -Sy wget #Arch
wget https://shell.xb6868.com/xbtools.sh && chmod 777 xbtools.sh
./xbtools.sh
```
